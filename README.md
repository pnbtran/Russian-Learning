## 🚀 Learning Workflow
This repository follows a structured learning workflow:

```text
📚 Resources
        │
        ▼
📖 Course
        │
        ▼
🎯 Practice
       ├─────────────┐
       ▼             │
🧠 Knowledge         │
       │             │
       ▼             │
📈 Progress      ❌ Mistakes
                     │
                     └──────► Practice
```

### Workflow Description
- **Resources**: Store original learning materials (Lưu trữ tài liệu học tập gốc).
- **Course**: Learn lesson by lesson (Học theo tài liệu).
- **Practice**: Apply what has been learned (Luyện tập và áp dụng kiến thức đã học).
- **Knowledge**: Organize long-term knowledge (Hệ thống hóa và lưu trữ kiến thức dài hạn).
- **Mistakes**: Record recurring mistakes and learn from them (Ghi lại lỗi và nguyên nhân).
- **Progress**: Track learning achievements over time (Tự theo dõi, đánh giá tiến độ).
